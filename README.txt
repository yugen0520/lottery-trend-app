走势编辑 App 说明

打开方式：
- 手机浏览器打开 index.html 或部署后的链接。
- 安卓 Chrome / Edge：点右上角菜单，选择“添加到主屏幕”。
- 苹果手机 Safari：点分享按钮，选择“添加到主屏幕”。
- 添加后桌面会出现“走势编辑”，像 App 一样打开。

功能：
1. 适合手机使用的走势数据编辑界面。
2. 可切换“排列三 / 福彩3D”。
3. 可新增一期数据，输入奖号后自动计算和值和百位。
4. 点击期次、奖号、和值单元格可直接编辑，回车保存。
5. 走势图是空白模板，只显示红点和走线。
6. 点击任意空白走势格，可改变该行红点位置，并自动重画折线。
7. 可切换 20/30/50 期显示，可删除最后一期。
8. 可开关画线。
9. 可导出/导入 JSON 数据，方便交给别人继续编辑。
10. 修改会自动保存在当前浏览器本地，下次打开仍保留。
11. 支持离线缓存，装到桌面后可当小 App 用。
